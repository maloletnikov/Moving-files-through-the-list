# Перемещение файлов по списку

## Описание

Скрипт перемещает файлы из папки `Источник` в папку `Назначение` согласно списку из файла `Список.txt`.

## Структура

```
Ваша_папка/
├── Список.txt
├── Источник/
│   ├── файл1.pdf
│   ├── файл2.pdf
│   └── ...
└── Назначение/
    └── (сюда попадут перемещённые файлы)
```

## Формат файла Список.txt

В файле указываются имена файлов **без расширения**, каждое с новой строки:

```
337204
364307
411903
437942
```

## Инструкция

1. Поместите `Список.txt` и папки `Источник`/`Назначение` в одну директорию
2. Откройте эту директорию в проводнике
3. Нажмите **ПКМ** → **Открыть в терминале** (или PowerShell)
4. Скопируйте и вставьте команду:

```powershell
Get-Content Список.txt | ForEach-Object { Move-Item "Источник\$_.pdf" "Назначение\" -ErrorAction SilentlyContinue }
```

5. Нажмите **Enter**

## Примечания

- Команда ищет файлы с расширением `.pdf`. Если расширение другое — замените `.pdf` в команде
- Если файл не найден — он просто пропускается (ошибки не выводятся)
- Файлы **перемещаются**, а не копируются
```
