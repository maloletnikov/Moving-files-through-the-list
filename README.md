# Перемещение файлов по списку (PowerShell)

## Описание

Автоматическое перемещение файлов из папки `Источник` в папку `Назначение` на основе списка имён, указанного в файле `Список.txt`.

## Структура проекта

```
Рабочая_папка/
├── Источник/          # Файлы, которые нужно переместить
├── Назначение/        # Папка, куда будут перемещены файлы
├── Список.txt         # Список имён файлов (по одному в строке)
└── README.md          # Эта инструкция
```

## Инструкция по использованию

1. Подготовьте папки `Источник` и `Назначение`, а также файл `Список.txt` в одной директории.

2. В файле `Список.txt` запишите имена файлов по одному в строке. Пример:

```
document_01
report_2025
invoice_march
```

3. Откройте терминал PowerShell в этой директории:  
   **ПКМ** на пустом месте в папке → `Открыть в терминале` (или `PowerShell`)

4. Вставьте и выполните следующую команду:

```powershell
Get-Content "Список.txt" | ForEach-Object { Move-Item -Path "Источник\$_.pdf" -Destination "Назначение\" -ErrorAction SilentlyContinue }
```

## Пояснение к команде

| Команда | Описание |
|---------|----------|
| `Get-Content "Список.txt"` | Читает файл построчно |
| `ForEach-Object { ... }` | Выполняет блок кода для каждой строки |
| `Move-Item -Path ... -Destination ...` | Перемещает файл из источника в назначение |
| `$_` | Автоматическая переменная, содержащая текущую строку из списка |
| `-ErrorAction SilentlyContinue` | Скрывает ошибки (например, если файл не найден в источнике) |

## Важные замечания

- **Расширение файлов:** В команде жёстко указано `.pdf`. Если ваши файлы имеют другие расширения или расширения уже прописаны в `Список.txt`, замените `"Источник\$_.pdf"` на `"Источник\$_"`.

- **Безопасный запуск:** Перед реальным перемещением добавьте параметр `-WhatIf`, чтобы увидеть план действий без фактического изменения файлов:

```powershell
Get-Content "Список.txt" | ForEach-Object { Move-Item -Path "Источник\$_.pdf" -Destination "Назначение\" -WhatIf -ErrorAction SilentlyContinue }
```

- **Перезапись:** Если в папке `Назначение` уже существует файл с таким же именем, он будет перезаписан без подтверждения.

- **Резервная копия:** Всегда рекомендуется скопировать папку `Источник` перед запуском массовых операций.
