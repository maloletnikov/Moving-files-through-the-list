# Перемещение файлов по списку

## Структура папок

```
📁 Ваша_папка/
│
├── 📄 Список.txt
│
├── 📁 Источник/
│   ├── 📄 337204.pdf
│   ├── 📄 364307.pdf
│   ├── 📄 411903.pdf
│   └── 📄 ... (другие файлы)
│
└── 📁 Назначение/
    └── (сюда попадут скопированные файлы)
```

## Формат Список.txt

```text
337204
364307
411903
437942
442697
476155
499431
519827
524239
557762
573991
578990
590532
```

## Как использовать

1. Откройте папку, где лежат `Список.txt`, `Источник` и `Назначение`
2. Нажмите **ПКМ** → **Открыть в терминале**
3. Выберите нужную команду и нажмите `Enter`:

### Перемещение файлов

```powershell
Get-Content Список.txt | ForEach-Object { Move-Item "Источник\$_.pdf" "Назначение\" -ErrorAction SilentlyContinue }
```

### Копирование файлов

```powershell
Get-Content Список.txt | ForEach-Object { Copy-Item "Источник\$_.pdf" "Назначение\" -ErrorAction SilentlyContinue }
```

## Важно

- Файлы должны быть с расширением `.pdf`
- Если расширение другое — замените `.pdf` в команде
- Несуществующие файлы пропускаются без ошибок
- `Move-Item` — **перемещает** файлы (удаляет из источника)
- `Copy-Item` — **копирует** файлы (оставляет в источнике)
